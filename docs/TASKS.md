# MVP TASKS

## Phase 1 — Foundation

Goals:
- Setup Next.js 15 app
- Setup TypeScript
- Setup Tailwind CSS
- Setup shadcn/ui
- Setup Supabase
- Setup authentication
- Setup OpenRouter API integration
- Build modern AI chat UI
- Responsive layout
- Sidebar navigation

Deliverable:
A working ChatGPT-like interface.

---

## Phase 2 — Core AI Features

Goals:
- Conversation storage
- Chat history
- Streaming AI responses
- Multi-model support
- File uploads
- Markdown rendering
- Conversation summaries
- Basic memory tagging

Deliverable:
Users can chat, upload files, and save conversations.

---

## Phase 3 — Memory System

Goals:
- Memory search
- Related memories
- Context restoration
- Automatic memory recall
- Memory graph generation
- Topic extraction
- Entity extraction

Deliverable:
AI can retrieve relevant past context automatically.

---

## Phase 4 — Permanent Storage

Goals:
- Arweave uploads
- Client-side encryption
- Permanent storage pipeline
- Upload queue
- Transaction tracking
- Storage status UI

Deliverable:
Conversations become permanently stored and privately encrypted.

---

## Phase 5 — AO Integration

Goals:
- AO memory indexing
- Retrieval optimization
- Memory registry
- User memory pointers
- Lightweight AO coordination

Deliverable:
AO-powered memory retrieval layer.

## Important Constraints

DO NOT:
- Build tokens
- Build DAO systems
- Overengineer architecture
- Build autonomous agents
- Build decentralized everything

FOCUS ON:
- Fast MVP development
- Clean UX
- Simple architecture
- AI memory experience
